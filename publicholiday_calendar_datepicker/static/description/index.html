Public Holiday Calendar/Datepicker

Enhance Your Odoo Experience with Holiday Visualization

Effortlessly identify holidays with our color-coded calendar and datepicker.

Key Features:

  • Color-coded holidays in the Odoo calendar for easy identification.
  • Holidays displayed in red, ensuring they stand out in the calendar view.
  • Improved holiday visibility in the datepicker for date selection.

Getting Started:

  1. Install the "Public Holiday Calendar/Datepicker" module in your Odoo instance.
  2. Navigate to the "Time Off" module.
  3. In the "Configuration" menu, select "Public Holidays."
  4. Set up and configure your public holidays.
  5. With public holidays configured, the color-coded calendar and datepicker will now reflect these holidays.
